### Prerequisites
- Flutter SDK (3.0 or higher)
- Dart SDK (3.0 or higher)
- An IDE (VS Code, Android Studio, etc.)

### Installation Steps

1. **Clone the repository**

2. **Navigate to project directory**
